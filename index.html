Sincronización entre dispositivos

Para sincronizar tus series entre dispositivos necesitás un proyecto gratuito de Supabase. Creá uno, y en Project Settings → API copiá la Project URL y la anon public key. Después, en el SQL Editor del proyecto, pegá y ejecutá esto una vez: 

create table public.bitacora (
  user_id uuid references auth.users on delete cascade primary key,
  data jsonb not null default '{}'::jsonb,
  updated_at timestamptz not null default now()
);
alter table public.bitacora enable row level security;
create policy "select own" on public.bitacora for select using (auth.uid() = user_id);
create policy "insert own" on public.bitacora for insert with check (auth.uid() = user_id);
create policy "update own" on public.bitacora for update using (auth.uid() = user_id);

Además, en Authentication → Providers → Email, desactivá "Confirm email" para no depender de correos de verificación en un uso personal.

Iniciá sesión (o creá una cuenta) para sincronizar. Usá el mismo email/contraseña en todos tus dispositivos.

API key de TMDB

Necesitás una API key gratuita de TMDB. Sacala en themoviedb.org/settings/api (cuenta gratis, pedís una "API Key" tipo v3, es inmediato) y pegala aquí. Se guarda solo en tu navegador.
